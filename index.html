The course acquires knowledge of programming technologies integrated with advanced sites, with emphasis on rich interactive experience, and accessible Accessibility Web site development and responsive web sites. To build such a site requires the ability to design a site properly, knowledge of client side and server side development technologies.

On the client side this includes: CSS3, HTML5, JavaScript and knowledge of libraries belonging to leading companies such as Bootstrap Twitter ./ React On the server side this includes database management knowledge, establishing a multi-client web server, developing components such as user registration, Authentication, Dynamic Content Management, Database Search, Shopping Basket Management. We will learn to develop based on -View-Model architecture .Node.JS such as (front-end / backend frameworks) in Controller operating systems
